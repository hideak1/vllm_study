# Part 2: LLMサービングシステム設計

**目標:** 特定のフレームワークに踏み込む前に、LLMサービングシステムの全体アーキテクチャを理解する。

このパートでは、システムアーキテクトのように考える力を養います。LLMサービングシステムのすべてのコンポーネントと設計上のトレードオフを理解することで、vLLMやSGLangのソースコードを読む際に、各コンポーネントが*なぜ*存在するのかがわかるようになります。
