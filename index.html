    Curriculum¶

    A systematic path from fundamentals to source-level mastery.

    Part 1: Foundations Transformer internals, GPU memory hierarchy, roofline analysis, batching strategies, quantization, and serving metrics. 8 chapters

    Part 2: System Design Request lifecycle, scheduler design, KV-cache memory management, model parallelism, and API layer architecture. 7 chapters

    Part 3: vLLM Architecture Full source code walkthrough — LLMEngine, Scheduler, PagedAttention, Block Manager, Worker, and API Server. 10 chapters

    Part 4: vLLM Internals KV-cache memory layout, prefix caching, CUDA graph capture, custom kernels, speculative decoding, and LoRA serving. 8 chapters

    Part 5: SGLang Architecture Runtime & Router, RadixAttention, Scheduler, ModelRunner, FlashInfer integration, and Frontend DSL. 10 chapters

    Part 6: SGLang Internals Advanced RadixAttention, data & expert parallelism, torch.compile, overlap scheduling, and FP8 quantization. 8 chapters

    :material-gpu: Part 7: CUDA & Triton Write GPU kernels from scratch — PagedAttention, FlashAttention, fused ops, quantized matmul, and CUTLASS. 8 chapters

    Part 8: New Models End-to-end guide to adding new model architectures to vLLM and SGLang with testing and performance tuning. 8 chapters

    Part 9: Production Deployment options, load balancing, monitoring, benchmarking methodology, cost optimization, and multi-node serving. 8 chapters

    Part 10: Comparison Systematic vLLM vs SGLang comparison — architecture, scheduling, memory management, and decision framework. 6 chapters

    Part 11: Contributing Dev environment setup, testing framework, debugging, performance profiling, and code review workflow. 7 chapters

    Part 12: Research Cutting-edge topics — disaggregated serving, KV-cache compression, long-context, reasoning models, and MoE optimization. 8 chapters
